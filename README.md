Morse Code keyboard made by Frederick

How to Use:

Morse code is a language consisted of dots and underscores(dishes), which is used to signal letters

Dots(.) are typically shorter than dashes should be held for around 1 second
Dishes(_) are longer than dashes and should be held for around 3 seconds
Spaces are longer and should be held around 7 seconds

Push the button and hold it for 1 second, 3 second, or 7 seconds to type out something
